9) server/index.js endpoints (summary)
-----------------------------
- GET /api/parking-lot: returns total spots, occupiedSpots, etc.
- POST /api/parking-lot/toggle/:spotNumber: toggles occupancy array and returns new state.
- GET /api/cctv: returns cctv cameras data.
- GET/POST /api/alerts: manage alerts.
- GET /api/activity-history: returns history.
- GET /api/stats: aggregated stats.
